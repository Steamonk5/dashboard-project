Finance Dashboard Template

A clean, responsive finance dashboard template built with React, TailwindCSS, and Vite. This template is designed for modern web applications with a focus on usability, responsiveness, and dark mode support.

Features

Built with React + Vite

TailwindCSS for modern, responsive styling

Dark mode with local storage support

Fully responsive (desktop + mobile)

Clean design with reusable components

Mobile-friendly menu

Works with JSON data for easy integration

Easy to customize and extend
